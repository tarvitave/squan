🚀 Meet Squan — the open source multi-agent AI development command center.

Squan lets you dispatch multiple AI agents simultaneously, each working in its own isolated git branch. No conflicts. No waiting. Just ship faster.

🔥 KEY FEATURES
• Multi-agent orchestration — run 4, 8, or 20 agents at once
• Visual Kanban board — track every agent's progress
• Real-time agent chat — follow up and redirect agents
• 1,000+ MCP extensions — Postgres, Slack, Jira, GitHub & more
• Multi-model support — Claude, GPT, Gemini, Ollama
• Skills system — chain tasks (Test → Fix → PR)
• Scheduler — cron jobs & event-driven automations
• Cost tracking — per-agent spend breakdown
• Git isolation — each agent gets its own worktree
• Claude Code integration — built-in terminal panel

🛠️ TECH STACK
• Frontend: React 19 + TypeScript + Tailwind + Zustand
• Backend: Express + TypeScript + SQLite (libsql)
• Desktop: Electron 33
• Agents: Direct API calls (Anthropic, OpenAI, Gemini, Ollama)
• Extensions: Model Context Protocol (MCP)

📋 REQUIREMENTS
• Windows 10+ or macOS (Apple Silicon)
• Anthropic API key (or OpenAI/Gemini/Ollama)
• Git installed
• Node.js 18+ (for development)
