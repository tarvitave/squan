🚀 Introducing Squan — the multi-agent AI development command center

I've been building something I think every dev team needs: a desktop app that lets you run multiple AI agents in parallel, each working on their own isolated git branch. No conflicts. No waiting. Just ship.

Here's what makes Squan different:

🤖 **Multi-agent orchestration** — Dispatch 4, 8, or 20 agents simultaneously. Each gets its own git worktree so they never step on each other.

📋 **Visual Kanban board** — See every agent's progress at a glance. Drag tasks between columns. Follow up with agents in real-time chat.

🔌 **1,000+ MCP extensions** — Connect to Postgres, Slack, Jira, GitHub, and more through the Model Context Protocol.

🧠 **Multi-model support** — Use Anthropic Claude, OpenAI GPT, Google Gemini, or run local models with Ollama. Switch providers per agent.

⚡ **Skills & Automations** — Chain agent tasks into workflows (Test → Fix → PR) and schedule them on cron or event triggers.

💰 **Cost tracking** — Per-agent cost breakdown so you always know what you're spending.

It's fully open source (Apache 2.0) and runs on both Windows and macOS.

📺 Check out the demo video to see it in action ⬆️
