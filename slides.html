The Problem

One terminal.
One task.
One bottleneck.

Claude Code is powerful — but running a single session means your AI works sequentially, just like a solo developer.

⏱

Sequential execution

Each task must complete before the next begins. No parallelism.

🧠

Context window limits

Large projects overwhelm a single session's context.

👁

No visibility

Hard to track what the agent did, how long it ran, what it cost.

🔁

Manual repetition

Security audits, design reviews, test runs — repeated manually every time.

Developer Workflow

From idea to merged code

1
📋
Create Release Train
Write a description of the feature or task. This becomes the agent's instructions (CLAUDE.md).
2
🚀
Dispatch
Squansq spawns a WorkerBee in an isolated git worktree and auto-sends the kickoff message.
3
👁
Monitor
Watch the live terminal, view replay frames, or let Mayor Lee coordinate while you focus elsewhere.
4
✅
Auto-land
When the agent signals DONE:, the Release Train lands automatically. Review the branch and merge.
5
🔁
Repeat
Standbys let you re-run security reviews, design audits, and tests with a single click.
06 / 08
Key Features

Everything you need to run a dev team at scale

🐝
WorkerBees
Ephemeral Claude Code agents, each with a git worktree, CLAUDE.md task injection, and auto-kickoff. Up to 15 named agents per project.
🎛
Mayor Lee Orchestrator
An AI manager that coordinates WorkerBees via MCP tools — creating tasks, dispatching agents, monitoring progress, and landing work.
📊
Kanban Board
Visual workflow board tracking every task from Open → In Progress → Landed. One-click dispatch, re-dispatch for stalled work, automatic cancellation.
⎇
Git Worktree Isolation
Every WorkerBee gets its own branch and working directory. Agents never conflict. Failed agents don't pollute the main branch.
📡
Live Terminals
Full xterm.js terminals for every agent. Watch Claude think in real time, send messages, or view ANSI-stripped replay frames after completion.
⚡
Standbys
One-click dispatch for repeatable tasks — security reviews, design audits, performance checks. System templates included, create your own.
💰
Pay-as-you-go API
WorkerBees use your Anthropic API key directly, avoiding Claude Pro subscription limits. Built-in cost activity dashboard.
🔄
Session Replay
Every 30 seconds a frame is captured. Step through what an agent did, view completion notes, and diagnose zombie failures.
🏗
Multi-workspace
Multiple workspaces, each with projects, agents, and a dedicated Mayor Lee. Add projects by detecting git repos in your workspace directory.
